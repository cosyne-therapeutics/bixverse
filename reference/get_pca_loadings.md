# Get the PCA loadings

Get the PCA loadings

Get the PCA loadings from a `SingleCells`.

## Usage

``` r
get_pca_loadings(x)

# S3 method for class 'ScCache'
get_pca_loadings(x)
```

## Arguments

- x:

  An object to get PCA loadings from.

## Value

The PCA feature loadings from the object (if found).
