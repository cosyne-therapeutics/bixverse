# Calculate rapidbly Jaccard similarities between rows

Helper function to quickly calculate the Jaccard similarity between the
rows across the two matrices.

## Usage

``` r
rs_jaccard_row_integers(data_1, data_2)
```

## Arguments

- data_1:

  Integer matrix. The first matrix to compare.

- data_2:

  Integer matrix. The second matrix to compare.

## Value

The average Jaccard similarity.
