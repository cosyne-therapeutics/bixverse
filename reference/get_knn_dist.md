# Get the KNN distance measures

Get the KNN distance measures

Get the KNN distances from a `SingleCells`.

## Usage

``` r
get_knn_dist(x)

# S3 method for class 'ScCache'
get_knn_dist(x)

# S3 method for class 'SingleCellNearestNeighbour'
get_knn_dist(x)
```

## Arguments

- x:

  An object to get the kNN distances from.
