# Set the HVG genes

Set the HVG genes

Set the highly variable genes for a `SingleCells` class.

## Usage

``` r
set_hvg(x, hvg)

# S3 method for class 'ScMap'
set_hvg(x, hvg)
```

## Arguments

- x:

  An object to set the HVGs for

- hvg:

  String or integer. The names or indices of the highly variable genes.
