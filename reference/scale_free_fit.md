# Calculate the goodness of fit for a power law distribution.

Calculate the goodness of fit for a power law distribution.

## Usage

``` r
scale_free_fit(k, breaks = 50L, plot = FALSE)
```

## Arguments

- k:

  Numeric vector. The vector of node degrees.

- breaks:

  Integer. Number of breaks for fitting the data.

- plot:

  Boolean. Shall the log-log plot be generated.

## Value

The R2 value of of the goodness of fit.
