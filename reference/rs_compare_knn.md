# Helper to compare kNN graphs

Helper to compare kNN graphs

## Usage

``` r
rs_compare_knn(knn_mat_a, knn_mat_b)
```

## Arguments

- knn_mat_a:

  Integer matrix. The first kNN graph to compare.

- knn_mat_b:

  Integer matrix. The second kNN graph to compare.

## Value

Vector of number of overlaps per sample.
