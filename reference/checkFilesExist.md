# Check that files exist

Checkmate extension for checking if files exist in the directory.

## Usage

``` r
checkFilesExist(x, file_names)
```

## Arguments

- x:

  String. The directory to check the files for.

- file_names:

  String. Vector of names of the expected files in this directory.

## Value

`TRUE` if the check was successful, otherwise an error message.
