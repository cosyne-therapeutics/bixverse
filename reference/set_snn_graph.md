# Set/add KNN

Set/add KNN

Set the sNN graph for a `SingleCells` class.

## Usage

``` r
set_snn_graph(x, snn_graph)

# S3 method for class 'ScCache'
set_snn_graph(x, snn_graph)
```

## Arguments

- x:

  An object to add the KNN data to

- snn_graph:

  Igraph. The sNN graph for subsequent clustering.
